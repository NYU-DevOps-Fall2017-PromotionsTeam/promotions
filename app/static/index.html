RESTful API for accessing Promotion Data

  • GET /promotions - Returns a list all of the Active Promotions
  • GET /promotions/{id} - Returns the Pet with a given promot-id number Almost Like a "PromoCode"
  • POST /promotions - creates a new Promotion in the database
  • DELETE /promotions/{id} - deletes a Promotion record in the database
  • PUT /promotions/delete-all - deletes all Promotions from the database
